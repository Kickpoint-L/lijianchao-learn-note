client家族

clientWidth/clientHeight可视区域的宽度可高度,当被盒子调用的时候指的是盒子本身的宽和高.当被body/html调用的时候则指的是可视区域的宽高(如果存在调试窗口，则显示能看到的版面的宽高)。 clientX,光标距离可视区域左侧的距离(event调用). clientY,光标距离可是区域上部的距离(event调用). clientTop/clientLeft 盒子的border宽高(边框).

三大家族width/height的区别

clientWidth/clientHeight = (内容部分+padding)

offsetWidth/offsetHeight = 整个盒子的宽高(包括border)

scrollWidth/scrollHeight = 内容宽高（不包含border）(能被卷的东西，如果超出计算整个内容)

top和left

offsetTop/offsetLeft ,距离具有定位的父类的顶部和左边距(不包含父类的dorder).

scrollTop/scrollLeft:(盒子也可以调用，必须有滚动条).表示浏览器无法显示的部分.被卷进去的部分.

clientTop/clientLeft 值是border边框的宽高.

clientY/clientX:调用者event.光标距离浏览器可视区域的距离（左、上）。

事件冒泡

当一个元素的某个事件被触发时(比如点击),同一个事件在他的所有祖先元素中都会触发,我们称为事件的冒泡.这个事件从原始元素开始一直冒泡到DOM树的最上层. 什么是冒泡：子元素事件被触动，父盒子的同样的事件也会被触动。

阻止冒泡的兼容写法
 if (event){
       event.stopPropagation();
}else {
       event.cancelBubble = true;
}

addEventListenner(参数1，参数2，参数3) 调用者是：事件源.参数1：事件去掉on 参数2 ：调用的函数 参数3：可有可无。没有默认false.false情况下，支持冒泡。True支持捕获。

正则表达式

正则表达式，又称规则表达式。在代码中简写为regex/regexp/RE. 正则表通常用来检索/替换那些符合某些规则(模式)的文本. 可以通过正则表达式，从字符串中获取我们想要的特定部分。 主要应用对象是文本,如表单验证,高级搜索. 正则本身就是字符串,用来匹配字符串 使用test测试匹配结果,结果是boolean类型 . [^\n\r] 除了换行和回车之外的任意字符（“”不行）

\d [0-9] 数字字符

\D [^0-9] 非数字字符

\s [ \t\n\x0B\f\r] 空白字符

\S [^ \t\n\x0B\f\r] 非空白字符

\w [a-zA-Z_0-9] 单词字符 字母数字下划线（注册要求用户名） ** \W** [^a-zA-Z_0-9] 非单词字符

var reg = /a/;  //匹配包含'a'的字符串
var reg2 = /abc/;  // 匹配 包含'abc'连着的字符串
var reg3 = /[abc]/; // 匹配 包含'a'或'b'或'c'的字符串
var reg4 = /[0-9a-zA-B]/; // 简化匹配 
var reg5 = /[^abc]/; // 匹配除了包含'a'或'b'或'c'的字符串
var qqReg = /^[1-9][0-9]{4,9}$/; // ^以什么开头  $以什么结尾
var reg6 = /^a*$/; // .test('aaaaaa')true;     *    意思是任意多个,0个或者多个
var reg7 = /^a+$/; // .test('aaaaaa'); true  +      重复一次或更多次  (>=1)
var reg8 = /^a?$/; // .test('aaaaaa');false   ?        重复零次或一次   （0||1）  要么有 要么没有
var reg9 = /^a{3}$/; // .test('aaaaaa');false   {n}      具体次数;
var reg10 = /^a{3,}$/; // .test('aaaaaa');true   {n,}	重复n次或更多  (x>=n)
var reg11 = /^a{3,5}$/; // .test('aaaaaa');true   {n,m}重复出现的次数比n多但比m少 (n<=x<=m)
